## Define Version Control
Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later.

## What do you understand by Git?
Git is a DevOps tool used for source code management.

## Describe the process of committing a project to github repository
1- Open a new repositry on github.
2- Create new file on VS code and give it same name as your created repository.
3- Open terminal on VS code.
4- initialize git by putting "git init" on the terminal.
5- Add the files in your new local repository by using "git add ." . This stages them for the first commit. 
6- Commit the files that you've staged in your local repository by using "git commit -m "First commit".
